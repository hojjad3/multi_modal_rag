This repository contains our project focused on building a Multi-Modal Retrieval-Augmented Generation (RAG) system, where we developed a tool capable of processing both text and images simultaneously. Our goal is to improve how AI retrieves information from complex documents, such as reports containing charts and diagrams, to generate accurate and comprehensive answers. To achieve this, we integrated computer vision models with Large Language Models (LLMs), allowing the system to fully understand both visual and textual contexts together. Ultimately, this creates a much more effective and advanced approach compared to traditional retrieval systems that rely solely on reading text.
